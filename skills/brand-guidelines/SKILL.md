# Peruna Tours Brand Styling

## Overview

To access Peruna Tours' official brand identity and style resources, use this skill.

**Keywords**: branding, corporate identity, visual identity, post-processing, styling, brand colors, typography, Peruna Tours brand, visual formatting, visual design, travel, Peru, tours

## Brand Identity

**Company Name**: Peruna Tours
**Tagline**: "Get to know the real Peru"
**Core Focus**: Authentic cultural experiences and guided tours to Peru's historical sites

### Brand Values

- **Expertise**: Expert guides with deep knowledge of Peruvian history
- **Quality & Comfort**: Professional transportation, small groups, personalized experiences
- **Accessibility**: Tours designed for various fitness levels and age groups
- **Community Support**: Supporting local communities and employing local guides
- **Safety & Reliability**: Hassle-free, organized experiences

### Tone of Voice

Enthusiastic yet professional; inspirational without being overly casual. Language emphasizes discovery ("Embark," "Journey," "Uncover") and authenticity.

## Brand Guidelines

### Colors

**Primary Colors:**

- Primary Green: `#34E0A1` - Main brand accent, reviews badge, CTAs
- Primary Blue: `#4D65FF` - Interactive elements, links, buttons
- Dark: `#1A1A1A` - Primary text and dark backgrounds
- Light: `#FFFFFF` - Light backgrounds and text on dark

**Supporting Colors:**

- Light Gray: `#F5F5F5` - Subtle backgrounds, cards
- Mid Gray: `#6B7280` - Secondary text, captions

### Typography

- **Font Family**: Poppins (with Arial fallback)
- **Font Weights**: Light (300), Regular (400), Medium (500)
- **Note**: Fonts should be pre-installed in your environment for best results

## Features

### Smart Font Application

- Applies Poppins font throughout all text elements
- Uses weight variations for visual hierarchy (Light for body, Medium for headings)
- Automatically falls back to Arial if Poppins unavailable
- Preserves readability across all systems

### Text Styling

- Headings (24pt+): Poppins Medium (500)
- Subheadings: Poppins Regular (400)
- Body text: Poppins Light (300)
- Smart color selection based on background
- Preserves text hierarchy and formatting

### Shape and Accent Colors

- Primary CTAs and highlights use Primary Green (#34E0A1)
- Interactive elements and links use Primary Blue (#4D65FF)
- Maintains visual interest while staying on-brand

## Technical Details

### Font Management

- Uses system-installed Poppins font when available
- Provides automatic fallback to Arial
- No font installation required - works with existing system fonts
- For best results, pre-install Poppins font in your environment

### Color Application

- Uses RGB color values for precise brand matching
- Applied via python-pptx's RGBColor class
- Maintains color fidelity across different systems
